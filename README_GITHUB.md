# SnagTin

**Phần mềm chụp màn hình, quay video & vẽ chú thích cho Windows.** Bản clone gọn nhẹ của Snagit — chụp nhanh, chỉnh sửa ngay, lưu vào thư viện.

## ✨ Tính năng

### 📸 Chụp màn hình
- Chụp **vùng chọn**, **toàn màn hình** (hỗ trợ nhiều màn hình)
- **Chụp cửa sổ** — tự highlight cửa sổ dưới con trỏ
- **Chụp hẹn giờ** — đếm ngược 3s / 5s rồi tự chụp

### 🎥 Quay video màn hình
- Quay **vùng chọn** hoặc **toàn màn hình**, xuất file **MP4 (H.264)**
- Tùy chọn **quay kèm âm thanh micro**
- Thanh điều khiển nổi: Tạm dừng / Dừng / đồng hồ đếm giờ

### ✏️ Trình chỉnh sửa ảnh
- Công cụ vẽ: **mũi tên, chữ nhật, elip, bút vẽ, chữ, highlight, làm mờ (blur)**
- **Đánh số bước** (①②③), **bong bóng thoại (callout)**, **stamp** (chèn biểu tượng)
- **Tiêu điểm (spotlight)** — làm tối xung quanh để làm nổi vùng quan trọng
- **Cắt ảnh (crop)**, **zoom**, **hoàn tác/làm lại** nhiều bước
- **Express Styles** — bộ preset màu/đổ bóng/tô nền chỉ với 1 click

### 🗂️ Thư viện
- Lưu toàn bộ ảnh **và video** đã chụp, tìm kiếm & gắn tag
- **Trình phát video nhúng** ngay trong app

## ⬇️ Tải về & cài đặt

1. Vào mục **[Releases](https://github.com/doanleox/snapzhot/releases/latest)**
2. Tải file **`SnagTin-Setup-x.x.x.exe`**
3. Chạy file và làm theo trình cài đặt

> Yêu cầu: Windows 10 trở lên. Không cần cài Python — file đã đóng gói sẵn mọi thứ.

## 🚀 Cách dùng nhanh

App chạy ở **khay hệ thống** (system tray). Phím tắt mặc định:

| Phím | Chức năng |
|------|-----------|
| `Ctrl+Shift+A` | Chụp vùng chọn |
| `Ctrl+Shift+F` | Chụp toàn màn hình |
| `Ctrl+Shift+R` | Bật/tắt quay video |

Hoặc **chuột phải vào icon ở khay hệ thống** để mở menu đầy đủ.

## 🔄 Cập nhật

App tự kiểm tra phiên bản mới. Khi có bản mới, bấm **"Cập nhật"** để tải bản cài mới nhất.
